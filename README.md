# 여행 위시 보딩패스 v13

압축을 풀고 `index.html`을 브라우저에서 열면 실행됩니다.

## v13 변경 사항

### 보딩패스 PNG 캔버스 완전 재설계
- **2컬럼 필드 정렬 수정**: Flight/Seat, Date/Gate, Boarding/Booking이 각각 왼쪽/오른쪽 컬럼에 정확히 정렬됩니다
- 컬럼 사이에 구분선(vertical separator) 추가
- FLIGHT / SEAT 등 레이블을 대문자로 명확히 표시
- 여권 정보 섹션, 라우트 섹션, 필드 섹션, QR 섹션이 각각 명확히 분리됩니다
- ticket tear 점선 + 원형 노치 디자인 추가
- QR 배경 카드 추가로 가독성 향상

### 화면 보딩패스 모바일 폰트 확대
- 탑승객명: 18px → 21px
- 공항 코드(ICN/NRT): 38px → 44px
- 필드 값(편명·좌석 등): 17px → 19px
- 레이블: 12px → 13px
- 모바일(400px 이하)에서도 별도 크기 최적화

### 공유 버튼 정리
- "더 많은 앱으로 공유하기 (시스템 공유)" 버튼 제거
- SNS 버튼 4종만 유지: 트위터/X · 카카오톡 · 인스타그램 · 링크 복사

## PNG 저장 4종
- 이 보딩패스 PNG 저장 (첫 번째 탑승권)
- 전체 보딩패스 저장 (발급된 전체 탑승권)

## 주의
